# Case-Social-Connect
Repositório que servirá como 1º Checkpoint da disciplina de WebDev da FIAP

📌 Sobre o projeto
O SocialConnect é uma plataforma digital desenvolvida para conectar pessoas com interesses, objetivos e valores em comum. A proposta é facilitar a criação de conexões relevantes, promovendo networking de forma simples, rápida e eficiente.
A aplicação busca oferecer uma experiência intuitiva, permitindo que os usuários encontrem e interajam com outras pessoas de maneira prática e segura.
